## About

Gogs 镜像中没有数据库服务，但官方安装中没有提到数据库，因此额外安装数据库容器。

## 安装

Gogs 的编排非常容器，但安装向导一定要注意各种不同个URL和端口设置。参考如下  

- 数据库：建议选择 SQLite
- 域名：使用浏览器当前访问的 URL，去掉 http:// 前缀， 保留后面的部分
- 应用 URL：使用浏览器当前访问的 URL
- SSH 端口：采用绑定的服务器对应的端口
- HTTP 端口：不修改
