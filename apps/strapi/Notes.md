## Strapi

Strapi 支持多种数据库连接


### 安装

Strapi 第一次启动时，需要较长的时间 Installing dependencies。请耐心等待

### 快速入门

1. 增加一个数据集，假如名为为：dockers


2. 增加一个用户，设置角色为 public


3. 编辑 Public role，使之有权限访问 dockers 数据集


4. 访问下面的 URL 便可以获取数据

   ```
   # 获取所有数据
   http://IP:1337/dockers/1
   
   # 获取第一条数据
   http://IP:1337/dockers/1
   ```
